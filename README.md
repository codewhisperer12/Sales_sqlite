# Sales_sqlite
📊 SQL + Python Mini Project: Sales Analytics

Objective:
Use SQL inside Python to extract simple sales insights (total quantity sold, total revenue) and visualize the results using basic print outputs and a bar chart.

Deliverables:
✔ Python script / Jupyter Notebook
✔ SQLite database connection (sales_data.db)
✔ 1–2 SQL aggregation queries
✔ Output displayed using print()
✔ Simple bar chart using matplotlib

Tech Stack:
Python | SQLite | pandas | matplotlib

Concepts Covered:

SQL execution inside Python

Database connectivity with sqlite3

Data aggregation using SQL

Python–SQL integration

Basic data visualization

A beginner-friendly end-to-end mini project demonstrating how SQL and Python work together for data analysis 🚀
